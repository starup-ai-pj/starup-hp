# STARUP HP改修 要件定義書

**作成日：** 2026年3月26日

**対象サイト：** [https://www.starup01.jp/](https://www.starup01.jp/)

**依頼先：** 山下さん（デザイナー兼エンジニア）

**バージョン：** v4.0（最終版）

---

## 1. 改修の目的

1. **パフォーマンス改善**：トップページのローディング速度の改善を最優先とする
2. **採用・人事目的での情報開示強化**：候補者が「STARUPで働くイメージ」を持てるよう、コンテンツの質・量・導線を強化する
3. **事業・会社情報の充実**：トップページだけでは薄い事業情報を、Serviceページ新設・Aboutページ改修で補完する

---

## 2. ターゲット候補者像


| ターゲット     | 属性イメージ                                           |
| --------- | ------------------------------------------------ |
| エンジニア・PM層 | 大手企業・メガベンチャーでの経験を持ち、スピード感ある環境・社会的インパクトを求めて転職検討中  |
| ビジネスパーソン層 | 製造業・コンサルティングファーム出身で、AI・テクノロジーを活用した新しいアプローチに関心がある |


**共通訴求軸：** 京都発プレシリーズA・事業立ち上げフェーズからの裁量／製造・物流・小売業界のAI活用という社会的意義／少数精鋭（正社員10名規模）で個人の影響範囲が大きい

## 5. ページ別 詳細要件

---

### 5-1. トップページ（/）

### パフォーマンス改善（最優先）

- ヒーローセクションの動画・大型SVGアニメーション・ループテキストアニメーションを見直し、表示速度優先の実装へ変更
- サービス紹介・ニュースサムネイル画像をWebP形式・適切サイズに最適化 + Lazy Load徹底
- **目標：** PageSpeed Insights モバイルスコア 70点以上

### セクション構成


| #   | セクション    | 内容                           | 変更         |
| --- | -------- | ---------------------------- | ---------- |
| 1   | Hero     | キャッチコピー + サブコピー              | アニメーション軽量化 |
| 2   | Mission  | ミッション文                       | 現状維持       |
| 3   | Service  | サービス3本カード（各Service詳細ページへリンク） | リンク追加      |
| 4   | 採用訴求（新設） | 「一緒に働く仲間を募集しています」            | 新規追加       |
| 5   | News     | 最新ニュース3件                     | 現状維持       |
| 6   | Footer   | お問い合わせ導線                     | 現状維持       |


**採用訴求セクション（新設）の掲載項目：**

- セクションタイトル（例：「Join Us」）
- 現在募集中のポジション名（2〜3件、求人票詳細ページへのリンク付き）
- Culture・Interviewページへの導線カード
- 「Careerページへ」CTAボタン

### 5-2. Aboutページ（/about）

現状の会社概要情報を維持しつつ、沿革セクションを追加する。

### セクション構成


| #   | セクション   | 内容                      | 変更    |
| --- | ------- | ----------------------- | ----- |
| 1   | Mission | ミッション文・説明               | 現状維持  |
| 2   | 会社概要    | 会社名・住所・設立・代表者・事業内容等・子会社 | 子会社追加 |
| 3   | 沿革（新設）  | 会社の歩みをシンプルなタイムライン形式で表示  | 新規追加  |


**沿革セクションの掲載項目・デザイン方針：**

- 縦型タイムライン（年月 + 出来事の一行説明）
- 掲載内容例：設立・サービスローンチ・主要取引開始・資金調達・受賞等
- デコラティブにせず、テキスト主体のシンプルな表示

### 5-4. Career ハブページ（/recruit）

候補者がこのページにたどり着けば、カルチャー・メンバー・求人票のすべてに到達できるハブとして機能させる。

### セクション構成


| #   | セクション               | 内容                          | 変更   |
| --- | ------------------- | --------------------------- | ---- |
| 1   | ページヘッダー             | 「Careers」タイトル + リード文        | 現状維持 |
| 2   | 採用メッセージ             | 代表からのメッセージ（200〜400字）        | 強化   |
| 3   | 求める人物像              | 4つのValueに具体的行動例を追記          | 強化   |
| 4   | 数字で見るSTARUP         | 会社・環境を数字で表現                 | 新規追加 |
| 5   | 働く環境・福利厚生           | 勤務形態・待遇の詳細                  | 新規追加 |
| 6   | 募集職種一覧              | 求人カード（→ /recruit/jobs/[id]） | 改修   |
| 7   | Culture・Interview導線 | 各サブページへの誘導カード               | 新規追加 |
| 8   | Member導線            | メンバー紹介への誘導カード               | 新規追加 |
| 9   | フォトギャラリー            | チームの雰囲気写真                   | 現状維持 |
| 10  | 応募CTA               | 応募フォームへ                     | 追加   |


**数字で見るSTARUP（新設）の掲載項目：**
設立からの月数・メンバー人数（正社員+業務委託）・主要取引先数・事業領域数
※数値はSTARUP側で確認・提供

**働く環境・福利厚生（新設）の掲載項目：**

- 勤務形態：フルリモート可 / フレックスタイム制（コアタイム 11:00〜16:00）
- 休日：年間125日（土日祝・年末年始）
- 各種社会保険完備（雇用・健康・厚生年金）
- 賞与：年2回（業績連動）
- ストックオプション制度あり
- 借り上げ社宅制度あり
- 書籍・学習ツール購入補助
- リモートワーク環境支援
- 交通費支給（所定範囲内）

### セクション構成

| #   | セクション    | 内容                     |
| --- | -------- | ---------------------- |
| 1   | ヘッダー     | 写真（大）/ 名前 / 役職 / 入社年   |

**インタビュー質問軸（案）：**

1. 入社前のバックグラウンドは？
2. 転職を検討したきっかけは？

4. 現在の仕事内容・やりがいは？
5. STARUPで働いて変わったことは？
6. どんな人と一緒に働きたいですか？

**初期公開想定：** 3〜5名（エンジニア・ビジネス職から各1名以上）
**実装要件：** CMSから将来追加できる構造が望ましい（Notionベースで可能か要確認）
