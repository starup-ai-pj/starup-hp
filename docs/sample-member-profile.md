# 経歴

2015年に東京大学工学部を卒業後、大手IT企業にてソフトウェアエンジニアとしてキャリアをスタート。Webアプリケーションやモバイルアプリケーションのフロントエンド開発に従事し、ユーザー体験を第一に考えたプロダクト開発を推進。

2018年からはテックリードとして、チームマネジメントとアーキテクチャ設計を担当。スクラムマスターとしてアジャイル開発の導入を主導し、開発効率を30%向上させる成果を上げる。

## 専門分野

- **フロントエンド開発**: React、Next.js、TypeScript を用いたモダンなWeb開発
- **UI/UXデザイン**: ユーザー中心設計に基づいたインターフェース構築
- **チームマネジメント**: アジャイル開発手法を活用したプロジェクト推進

## プロジェクト実績

### Eコマースプラットフォーム刷新（2020-2021）
大手小売企業のECサイトを完全リニューアル。Next.jsとHeadless CMSを採用し、ページ表示速度を50%改善。年間売上20%増加に貢献。

### 社内業務システム開発（2019-2020）
従業員5000名規模の企業向け業務管理システムを開発。React + GraphQLのモダンスタックで構築し、業務効率化を実現。

### モバイルアプリケーション開発（2017-2018）
ヘルスケア分野のモバイルアプリを開発。React Nativeを採用し、iOS/Android両対応を実現。リリース3ヶ月で10万ダウンロードを達成。
